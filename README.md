# Environment Variables Task

## Task Overview
This task demonstrates how to work with environment variables in Linux.

## Steps Performed
1. Viewed environment variables using printenv and echo.
2. Created custom environment variables.
3. Modified the PATH variable.
4. Made variables persistent using .bashrc.
5. Tested changes after restarting terminal.
6. Debugged PATH issues using which and echo.

## Tools Used
- Linux (WSL)
- Bash shell
- nano editor
